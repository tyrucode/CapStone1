# CapStone1
CapStone Project using all of the technologies I have learned.

My Ideas for this project - 

1. Random Song Roulette (Spotify API)
An app where users can spin a "wheel" or press a button to get a completely random song from Spotify's vast library. Add filters, 
so users can specify a genre, decade, or mood if they want to narrow down their discovery.
Features:
Random song picker with optional filters (genre, mood, etc.).
Users can save songs they like directly to a playlist.
A fun, gamified design with a wheel or button for selecting songs.
I would also like to make it so users can randomize entire playlists based on these filters they decide on, and integrate it so it adds the
playlist directly to the users spotify account.

2.Concert & Events Playlist (Spotify / TicketMaster API)
Description: Build a tool that generates a playlist based on upcoming concerts or events near the user’s location. The app could pull in 
artists from the events and create a playlist so the user can familiarize themselves with the music before attending.
Features:
Users can enter their location and search for nearby events.
Generate a playlist with the most popular songs from the artists playing at these events.
Integration with event platforms (like Eventbrite or Ticketmaster) for more detailed event info.
If I did this one I would also use the ticketmaster api alongside it to get information about specific events
main issue is figuring out how to make a location / map sort of visual. 

3. Guess the Game Soundtrack (Spotify API)
Description: A game where players listen to short clips of video game soundtracks (might just do regular songs too) and try to guess the game they’re from. Using Spotify's API 
Features:
Play audio clips of game soundtracks (or any songs, maybe users can choose what genre/mood) and provide multiple-choice options.
Include a scoring system and time-based challenges for more excitement.
